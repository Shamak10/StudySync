# StudySync – A Study Planning Application

**StudySync** is a cross-platform study planning mobile application built using **.NET MAUI**. It helps students stay organized, manage their assignments, track progress, and achieve academic goals efficiently. Designed with productivity and simplicity in mind, this app is tailored for high school and university students balancing multiple classes.

---

## Features

- Add, edit, and delete tasks  
- Organize tasks by course  
- Set and manage weekly and long-term goals  
- Track task progress with visual graphs  
- Get personalized study tips  
- View what’s due today and upcoming tasks  
- Daily motivational quotes  
- Cross-platform support (Android/iOS)

---

## Target Audience

Primarily aimed at:

- University students  
- High school students  

This application is especially beneficial for students managing multiple courses and deadlines, helping them stay on top of their academic responsibilities.

* How to Use the App
Launch the app – View today's tasks, upcoming deadlines, and a motivational quote.

Add a Task – Use the + button to add a new task with course and due date.

Manage Tasks – Tap on a task to edit, mark as complete, or delete it.

Track Progress – Navigate to the Progress page to see completion stats via charts.

Set Goals

Weekly Goal – Set short-term targets for the week.

Major Milestone – Use the calendar to set long-term academic goals.

* Tech Stack & NuGet Packages
.NET MAUI (Multi-platform App UI)

C#, XAML, MVVM architecture

SQLite – Local data storage

ObservableCollection – Dynamic task lists

Microcharts – Visual progress representation

CommunityToolkit.MVVM, CommunityToolkit.Maui
